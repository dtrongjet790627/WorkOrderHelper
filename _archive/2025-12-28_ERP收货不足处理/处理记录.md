## 完整处理流程

### 第一步：读取ERP导出的Excel数据

**重要**：使用"完成数量"字段，而非"合格品入库数量"
- **完成数量**（Col13）：系统触发自动收货的数量
- **合格品入库数量**（Col7）：审核后的操作数量（可能与完成数量不同）

```python
import pandas as pd
df = pd.read_excel('D:/ACC运维记录/TEMP/生产汇报单_xxx.xlsx')
df = df.iloc[:-1]  # 排除合计行

wono = df.iloc[0, 4]   # 源单编号（工单号）
partno = df.iloc[0, 6] # 物料编码

# 按批号汇总 - 使用完成数量(Col13)
batch_data = {}
for idx, row in df.iterrows():
    batch = str(row.iloc[19])  # 批号
    qty = int(row.iloc[13])    # 完成数量（重要！不是Col7）
    batch_data[batch] = batch_data.get(batch, 0) + qty
```

### 第二步：对比并同步epr_report_work_history

```sql
-- 查询数据库中该工单按批次汇总
SELECT packid, COUNT(*) AS qty
FROM epr_report_work_history
WHERE wono = '<工单号>'
GROUP BY packid
ORDER BY packid;

-- 对比Excel数据，找出差异批次
-- 如果DB数量 > Excel数量，删除多余记录
DELETE FROM epr_report_work_history
WHERE ROWID IN (
    SELECT ROWID FROM (
        SELECT ROWID FROM epr_report_work_history
        WHERE wono = '<工单号>' AND packid = '<批次号>'
        ORDER BY report_time DESC
    ) WHERE ROWNUM <= <删除数量>
);
COMMIT;
```

### 第三步：对比并同步pack_history

**关键**：对比时必须基于ACC_WO_WORKORDER_DETAIL的主码和LINE字段

```sql
-- 找出pack_history中有（且属于该工单）但epr_report_work_history中没有的记录
DELETE FROM pack_history ph
WHERE EXISTS (
    SELECT 1 FROM acc_wo_workorder_detail awd 
    WHERE awd.unitsn = ph.unitsn 
    AND awd.wono = '<工单号>' 
    AND awd.status = 2
    AND awd.line = ph.line
)
AND NOT EXISTS (
    SELECT 1 FROM epr_report_work_history erh
    WHERE erh.unitsn = ph.unitsn
    AND erh.wono = '<工单号>'
);
COMMIT;
```

### 第四步：更新pack_info数量

```sql
-- 查找pack_history与pack_info数量不符的批次
SELECT 
    ph.packid, ph.actual_qty, pi.currquantity,
    ph.actual_qty - pi.currquantity AS diff
FROM (
    SELECT packid, COUNT(*) AS actual_qty
    FROM pack_history
    WHERE packid IN (
        SELECT DISTINCT ph2.packid FROM pack_history ph2
        WHERE EXISTS (
            SELECT 1 FROM acc_wo_workorder_detail awd 
            WHERE awd.unitsn = ph2.unitsn AND awd.wono = '<工单号>' AND awd.status = 2
        )
    )
    GROUP BY packid
) ph
JOIN pack_info pi ON ph.packid = pi.packid
WHERE ph.actual_qty != pi.currquantity;

-- 更新CURRQUANTITY
UPDATE pack_info 
SET currquantity = <正确数量>, lastupdatetime = SYSDATE 
WHERE packid = '<批次号>';
COMMIT;
```

### 第五步：补充缺失记录（以ACC_WO_WORKORDER_DETAIL为准）

#### 5.1 找出缺失记录
```sql
SELECT awd.unitsn, awd.line, awd.partno
FROM acc_wo_workorder_detail awd
WHERE awd.wono = '<工单号>' AND awd.status = 2
AND NOT EXISTS (
    SELECT 1 FROM pack_history ph
    WHERE ph.unitsn = awd.unitsn AND ph.line = awd.line
);
```

#### 5.2 确定目标批次（倒序第二个数量为0的批次）
```sql
SELECT * FROM (
    SELECT packid, currquantity, status,
           ROW_NUMBER() OVER (ORDER BY lastupdatetime DESC) AS rn
    FROM pack_info
    WHERE prodtype = '<型号>' AND currquantity = 0
) WHERE rn = 2;
```

#### 5.3 获取参考值（最近status=2且数量非0的批次）
```sql
SELECT packid, stn FROM (
    SELECT packid, stn, ROW_NUMBER() OVER (ORDER BY lastupdatetime DESC) AS rn
    FROM pack_info
    WHERE prodtype = '<型号>' AND status = 2 AND currquantity > 0
) WHERE rn = 1;
```

#### 5.4 插入缺失记录到pack_history
```sql
INSERT INTO pack_history (packid, unitsn, packbarcode, packdate, lastupdatetime, line, stn, trwsn, customerpackid, customerpartno)
SELECT 
    '<目标批次>' AS packid,
    awd.unitsn,
    awd.unitsn AS packbarcode,
    SYSDATE, SYSDATE,
    '<产线LINE>' AS line,
    '<参考STN>' AS stn,
    NULL, NULL, NULL
FROM acc_wo_workorder_detail awd
WHERE awd.wono = '<工单号>' AND awd.status = 2
AND NOT EXISTS (
    SELECT 1 FROM pack_history ph
    WHERE ph.unitsn = awd.unitsn AND ph.line = awd.line
);
COMMIT;
```

#### 5.5 更新pack_info
```sql
UPDATE pack_info
SET currquantity = <插入数量>,
    status = 2,
    lastupdate = SYSDATE,
    lastupdatetime = SYSDATE,
    stn = '<参考STN>',
    drag = (SELECT drag FROM pack_info WHERE packid = '<参考批次>'),
    generatorname = (SELECT generatorname FROM pack_info WHERE packid = '<参考批次>'),
    customerpackid = (SELECT customerpackid FROM pack_info WHERE packid = '<参考批次>'),
    customerpartno = (SELECT customerpartno FROM pack_info WHERE packid = '<参考批次>')
WHERE packid = '<目标批次>';
COMMIT;
```

### 第六步：验证

## Excel字段说明
| 列号 | 字段名 | 说明 |
|------|--------|------|
| Col4 | 源单编号 | 工单号(WONO) |
| Col6 | 物料编码 | 型号(PARTNO/PRODTYPE) |
| Col7 | 合格品入库数量 | 审核后的操作数量 |
| **Col13** | **完成数量** | **系统触发自动收货数量（应使用此字段）** |
| Col19 | 批号 | 批次号(PACKID) |

## 注意事项
1. **根据工单号前缀识别产线，选择正确的数据库用户**
2. **Excel读取时使用"完成数量"(Col13)，而非"合格品入库数量"(Col7)**
3. 对比时必须加LINE字段条件
4. 选择目标批次时取倒序第二个（避免选中正在生产的批次）
5. 所有操作需COMMIT提交
6. 处理完成后系统自动通过ERP_REPORT_VIEW_2025视图触发补收
